# MindMorph AI

Modern AI solutions platform built with Next.js and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- Next.js 13
- React 18
- Tailwind CSS
- ESLint
